# app-aiagent-builder

## 📝 概要・説明
ブラウザ上で直感的にAI（Gemini等）を活用したワークフローを構築・実行できる、ノードベースのWebアプリケーションです。

「ユーザー入力」「AI処理」「出力表示」といったノードをキャンバス上に自由に配置し、それらを線で繋ぐことで、データ処理のパイプラインを視覚的に作成できます。作成したワークフローは、内蔵のチャットUIから即座にテスト実行することが可能です。レスポンシブデザインとタッチイベントに完全対応しており、スマートフォンやタブレットからでも快適に操作できます。

## ✨ 主な機能一覧
- **直感的なノードエディタ**
  - キャンバス上でのノードのドラッグ＆ドロップ移動
  - ポート間のドラッグ操作による動的な結線（SVGベジェ曲線）
  - ワンクリックでのノードや結線の削除・全体クリア機能
- **3種類の専用ノード**
  - 🔵 **ユーザー入力**: チャットUIからの入力を受け取る起点ノード
  - 🟣 **AI処理**: システムプロンプト（目標や指示）を設定し、外部API経由でテキスト処理を実行するノード
  - 🟢 **出力表示**: 処理結果をチャット画面に返信する終点ノード
- **プレビュー・チャットUI**
  - 構築したワークフローの挙動を確認できるチャットインターフェース
  - 処理中はノード間の線がアニメーションし、データの流れを視覚化
  - メッセージの「縦書き表示」を取り入れたユニークで洗練されたデザイン
- **マルチデバイス対応**
  - PC環境では左右のペイン分割レイアウト
  - モバイル環境では「ワークフロー」「チャット」のタブ切り替えと、フローティングアクションボタン(FAB)による快適なUIを提供

## 🛠 使用技術・ライブラリ
本プロジェクトは、モダンなフロントエンド環境をフレームワークなし（Vanilla JS）の単一HTMLファイルで実現しています。

- **コア技術**: HTML5, CSS3, JavaScript (ES6+)
- **スタイリング**: Tailwind CSS (CDN)
- **アイコン**: Lucide Icons (CDN)
- **フォント**: Google Fonts (Inter)
- **グラフィックス**: SVG (動的な結線描画)
- **外部API連携**: Fetch APIを使用した非同期通信（Cloudflare Workers等でホストされたAIエンドポイントへリクエストを送信）

## 🚀 セットアップ・使い方

### セットアップ方法
本アプリケーションは単一のHTMLファイルで構成されているため、面倒な環境構築やビルド作業は不要です。

1. プロジェクトのHTMLファイルをダウンロードします。
2. Google Chrome、Safari、EdgeなどのモダンブラウザでHTMLファイルを直接開きます。
3. *※注意: AIの推論APIにアクセスするため、利用にはインターネット接続環境が必要です。また、必要に応じてコード内の `WORKER_URL` をご自身のAPIエンドポイントに変更してご使用ください。*

### 使い方
1. **ノードの配置**
   画面上部のツールバー（モバイルの場合は右下の `+` ボタン）から、ノード（ユーザー入力、AI処理、出力表示）をキャンバスに追加します。
2. **ワークフローの構築**
   - 各ノード上部をドラッグして配置を整理します。
   - 「AI処理」ノード内のテキストエリアに、AIに対するプロンプト（要約、翻訳、抽出などの指示）を記述します。
   - ノード下部の青いポート（出力）から、次のノード上部の緑のポート（入力）へドラッグして線を繋ぎます。
3. **テスト実行**
   - 画面右側のチャットパネル（モバイルは「チャット」タブ）の入力欄にメッセージを入力し、送信します。
   - 構築したフローに沿ってAIが処理を実行し、結果がチャット画面に返ってきます。
   - ヘッダーの「テスト実行」ボタンを押すことで、ダミーテキストによる即時テストも可能です。
