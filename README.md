# app-aiagent-builder

## 概要・説明
「AIエージェント - ワークフロービルダー」は、ブラウザ上で直感的にAIの処理フローを構築できるノードベースのWebアプリケーションです。

キャンバス上に「ユーザー入力」「AI処理」「条件分岐」「出力表示」といったノードを配置し、それらを線で繋ぐことで、プロンプトチェーンや条件に応じたAIの応答フローをノーコードで作成できます。作成したワークフローは、画面右側（モバイルではタブ切り替え）のチャットUIからすぐにテスト実行することが可能です。

PCだけでなくスマートフォンでのタッチ操作・スワイプにも完全に最適化されたレスポンシブデザインを採用しており、いつでもどこでもアイデアを形にすることができます。

## 主な機能一覧
- **ノードベースの視覚的エディタ**
  - キャンバス上でのノード追加、ドラッグ＆ドロップ移動、ワンクリック削除
  - 余白のドラッグによるキャンバスのパン（画面移動）操作
- **4種類の専用ノード**
  - **ユーザー入力**: チャットからのメッセージを受け取る起点
  - **AI処理 (Gemini等)**: カスタムプロンプトを設定し、AIにテキスト処理を実行させる
  - **条件分岐**: 前のノードの出力結果に特定の文字列が「含まれる/含まれない」を判定し、True/Falseで処理をルーティング
  - **出力表示**: 最終的な結果をチャット画面に返信する終点
- **インタラクティブな結線（エッジ）**
  - ノードのポート間をドラッグ（またはタッチ）して直感的に接続
  - 条件分岐時の経路を視覚的に分かりやすく色分け（Trueは緑、Falseは赤）
  - 結線された線をタップ・クリックして簡単に削除
- **プレビュー・チャット環境**
  - 構築したワークフローを即座にテストできる組み込みのチャットUI
  - 実行中の通信ステータス表示や、処理が通過している経路（エッジ）のアニメーションハイライト
- **クロスプラットフォーム最適化**
  - スマートフォン向けに最適化されたタップ領域の拡大、スクロール制御、タブ切り替えUIの実装

## 使用技術・ライブラリ
- **フロントエンド（フレームワーク非依存）**
  - HTML5 / CSS3 / Vanilla JavaScript
  - SVG (動的なベジェ曲線による結線描画)
- **UI / スタイリング (CDN経由)**
  - Tailwind CSS: ユーティリティファーストの柔軟なデザイン
  - Lucide Icons: 軽量でモダンなSVGアイコン
  - Google Fonts (Inter): 視認性の高いタイポグラフィ
- **バックエンド / API連携**
  - Fetch APIによる外部AI API（Cloudflare Workers環境）への非同期通信

## セットアップ・使い方

### セットアップ
本プロジェクトはフロントエンドのコードが単一のHTMLファイルにまとまっており、ローカル環境での特別なビルドプロセスやサーバーの立ち上げは不要です。

お手元の環境にHTMLファイルを保存し、Google Chrome、Safari、EdgeなどのモダンWebブラウザで開くだけですぐに利用を開始できます。

### 使い方
1. **ワークフローの構築**
   - 画面上部（モバイルは上段）のツールバーから追加したいノードのボタンをクリックします。
   - 配置されたノードの上部ヘッダーをドラッグして、好きな位置に配置します。
2. **ノードの接続**
   - 各ノードの下部にある丸いポートから、次のノードの上部にあるポートへ線をドラッグ（スワイプ）して繋ぎます。
3. **ノードの詳細設定**
   - **AI処理ノード**: テキストエリアに「文章を要約して」「英語に翻訳して」などのAIに対するプロンプト（指示）を入力します。
   - **条件分岐ノード**: ドロップダウンから条件を選び、判定の基準となるキーワードを入力します。
4. **テスト実行**
   - 画面右側（モバイルの場合は「チャット」タブに切り替え）の入力欄からメッセージを送信します。
   - ワークフローに沿って処理が順番に実行され、AIの応答や分岐の結果がチャット画面に表示されます。
   - 右上の「テスト実行」ボタンを使えば、ダミーテキストを入力して即座に動作確認を行うことも可能です。
5. **キャンバスの初期化**
   - ツールバーの「クリア」ボタンを押すことで、配置したノードやチャットの履歴をすべてリセットできます。
