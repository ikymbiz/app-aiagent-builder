# app-aiagent-builder

## 概要・説明
直感的なノードベースのUIを用いて、AIエージェントの応答フローを構築・テスト・公開できるブラウザ完結型のWebアプリケーションです。

キャンバス上でノード（処理ブロック）を繋ぎ合わせることで、プログラミングの知識がなくても複雑なAIの振る舞いや条件分岐を設定できます。統合されたチャットUIを用いてその場で動作テストができるほか、完成したワークフローは単一のWebアプリ（HTMLファイル）として書き出され、画面上から直接GitHub Pagesへデプロイすることが可能です。

## 主な機能一覧
- **直感的なノードエディタ**
  - キャンバスのパン（移動）やノードのドラッグ＆ドロップ配置
  - SVGによる滑らかな結線アニメーションと実行時のハイライト表示
- **4種類の専用ノード**
  - `ユーザー入力`: チャットからのメッセージを受け取る起点
  - `AI処理`: システムプロンプトや目標を設定し、AIに回答を生成させる
  - `条件分岐`: 特定のキーワードが含まれているかで処理をYes/Noに分岐
  - `出力表示`: チャット画面に最終的な結果を返信
- **統合テスト環境**
  - 作成したワークフローを即座に試せるチャットUIを搭載（モバイルではタブ切り替えで対応）
- **柔軟なAI連携**
  - Gemini APIキーを入力しての直接呼び出し
  - Cloudflare Workerをプロキシとして利用するエンドポイントに対応
- **データ管理**
  - ローカルストレージへの自動保存・手動復元機能
  - ワークフローデータのJSON形式でのエクスポート/インポート
- **ワンクリック・デプロイ機能**
  - GitHubのPersonal Access Tokenを使用して、作成したAIアプリをGitHub Pagesへ自動デプロイ

## 使用技術・ライブラリ
本プロジェクトは、ビルド環境を必要としないピュアなフロントエンド技術で構成されています。

- **コア言語**: HTML5, CSS3, JavaScript (Vanilla JS)
- **UI/スタイリング**: [Tailwind CSS](https://tailwindcss.com/) (CDN経由)
- **アイコン**: [Lucide Icons](https://lucide.dev/) (CDN経由)
- **フォント**: Google Fonts (Inter)
- **外部API**: 
  - Google Gemini API (AIテキスト生成用)
  - GitHub REST API (リポジトリ作成・ファイルアップロード・Pages設定用)
- **描画**: HTML DOM (ノード) + SVG (結線)

## セットアップ・使い方

### セットアップ（起動方法）
ビルドやサーバーの立ち上げは不要です。
1. 提供されているコードを `index.html` として保存します。
2. Google Chrome、Edge、Safariなどのモダンブラウザで `index.html` を開くだけで即座に使用可能です。

### 基本的な使い方

**1. AI APIの初期設定**
- 画面右上（ヘッダー）の「歯車アイコン（設定）」をクリックします。
- 「Gemini API (直接呼び出し)」を選択し、ご自身のGemini APIキーを入力して保存します。（※キーはブラウザにのみ保存され、外部サーバーには送信されません）

**2. ワークフローの構築**
- **追加**: キャンバスの何もない場所で「右クリック」（スマートフォンの場合は「長押し」）するとメニューが開き、ノードを追加できます。
- **結線**: ノードの上下にあるポート（色付きの丸）から、他のノードのポートへドラッグ＆ドロップして処理を繋ぎます。
- **設定**: 「AI処理」ノードのテキストエリアにプロンプト（指示）を、「条件分岐」ノードに判定用キーワードを入力します。

**3. テスト実行**
- 画面右側（スマートフォンの場合は「チャット」タブ）のプレビュー画面の入力欄からメッセージを送信します。
- 構築したワークフローの経路に沿ってAIが処理を行い、応答が返ってくるのを確認できます（処理中の線はハイライトされます）。

**4. 保存と読み込み**
- ヘッダーの「フォルダアイコン」から、ブラウザ内への手動保存、またはファイル（.json）としての出力・読み込みが可能です。

**5. GitHubへのデプロイ**
- ヘッダーの「雲アイコン（デプロイ）」をクリックします。
- `repo`権限を持ったGitHub Access Tokenと、任意の「リポジトリ名」を入力し「デプロイを実行」をクリックします。
- 自動的に独立したAIチャットアプリ（HTML）が生成され、あなたのGitHub Pagesとして世界中に公開されます。
