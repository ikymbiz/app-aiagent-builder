# app-aiagent-builder

## 概要・説明
ブラウザ上で直感的にAIの処理フローを構築・テストできる、ノードベースのワークフローエディタです。
「ユーザーからの入力受け取り」「AI（Gemini等）によるプロンプト処理」「結果のチャット出力」といった一連のプロセスを、視覚的に繋ぎ合わせて簡単にAIエージェントを作成できます。
PCおよびモバイルデバイスの双方に最適化されたレスポンシブデザインを採用しており、スマートフォンからでもタッチ操作で快適にワークフローを構築可能です。

## 主な機能一覧
- **ノードベースのエディタ**: キャンバス上でノードを自由に配置し、ドラッグ＆ドロップでノード同士を接続してフローを定義します。
- **3種類の基本ノード**:
  - **ユーザー入力**: チャットからのメッセージを受け取る起点ノード。
  - **AI処理**: カスタムプロンプト（指示・目標）を設定し、受け取ったデータをAIに処理させるノード。
  - **出力表示**: AIの処理結果をチャット画面に返信する終点ノード。
- **リアルタイム・プレビュー**: 構築したワークフローを画面右側（モバイルでは「チャット」タブ）のチャットUIですぐにテスト実行できます。
- **実行アニメーション**: ワークフロー実行時、データの流れに合わせて結線（エッジ）がハイライトされ、処理の進行を視覚的に確認できます。
- **モバイル・タッチ対応**: スマートフォンやタブレットでのタッチ操作（ドラッグや結線）に完全対応。モバイル環境ではタブUIでエディタとチャットをスムーズに切り替えられます。

## 使用技術・ライブラリ
- **コア技術**: HTML5, CSS3, Vanilla JavaScript (フロントエンドフレームワーク不使用)
- **スタイリング**: Tailwind CSS (CDN経由)
- **アイコン**: Lucide Icons (CDN経由)
- **フォント**: Google Fonts (Inter)
- **グラフィック**: SVG (ノード間の動的な結線描画)
- **外部API通信**: Fetch API (Cloudflare Workersなどの外部バックエンドエンドポイントを利用してAI処理を実行)

## セットアップ・使い方

### セットアップ
本プロジェクトは単一のHTMLファイルで構成されているため、Node.jsなどのビルド環境は一切不要です。
1. 提供されたHTMLコードを任意の名前（例: `index.html`）で保存します。
2. 保存したファイルをGoogle Chrome、Safari、Edgeなどのモダンブラウザで開くだけで、すぐに利用を開始できます。

### 使い方
1. **ノードの追加**
   画面上部のツールバーから「ユーザー入力」「AI処理」「出力表示」のボタンをクリック（タップ）し、キャンバスにノードを配置します。
2. **ワークフローの構築**
   各ノードの右側にあるポート（出力）から、次のノードの左側にあるポート（入力）へドラッグして線を繋ぎます。
   基本的な流れは `ユーザー入力` → `AI処理` → `出力表示` となります。
3. **プロンプトの設定**
   「AI処理」ノード内のテキストエリアに、AIに対する具体的な指示や役割（例：「文章を要約してください」など）を入力します。
4. **テスト実行**
   画面右側の「プレビュー・チャット」（モバイルの場合は画面下の「チャット」タブ）からメッセージを入力して送信すると、構築したワークフローに沿ってAIが処理を行い、応答が返ってきます。
5. **編集と削除**
   - **ノードの移動**: ノードの上部ヘッダー部分をドラッグして移動させます。
   - **ノードの削除**: ノード右上にある「×」ボタンを押します。
   - **結線の削除**: 繋いだ線（エッジ）をクリック（タップ）すると削除確認ダイアログが出ます。
   - **キャンバスのリセット**: 画面右上の「クリア」ボタンを押すと、すべてのノードとチャット履歴がリセットされます。
